## Description
This project demonstrates a cloud-based solution using Microsoft Azure
to securely store documents, analyze them using AI services, and manage
user access.

## Azure Services Used
- Azure App Service
- Azure Blob Storage
- Azure SQL Database
- Azure Cognitive Services
- Azure Active Directory

## Architecture
The user accesses the application hosted on Azure App Service.
Documents are stored in Azure Blob Storage.
AI services analyze the documents.
Security is handled using Azure Active Directory.

## Security
Role-based access control and encryption are used to protect data.

- AZ-900 Azure Fundamentals
- AI-900 Azure AI Fundamentals
- SC-900 Security Fundamentals
